## 配置说明

### 基本配置
创建配置文件 `config.yaml`:
```yaml
# 基本配置
general:
  name: "My Project"
  version: "1.0.0"

# 网络配置
network:
  host: "localhost"
  port: 8080
  ssl: true

# 数据库配置
database:
  type: "postgresql"
  host: "localhost"
  port: 5432
  name: "mydb"
```

### 环境变量
```bash
# 设置环境变量
export API_KEY="your-api-key"
export DATABASE_URL="postgresql://user:pass@localhost/mydb"
```

### 配置文件位置
- Linux/macOS: `~/.config/tool/config.yaml`
- Windows: `%APPDATA%\tool\config.yaml`

## 故障排除

### 常见问题

#### 问题1: 安装失败
**症状**: 安装过程中出现错误
**解决方法**:
1. 检查网络连接
2. 验证系统权限
3. 查看详细日志
```bash
tool install --log-level debug
```

#### 问题2: 服务无法启动
**症状**: 服务启动后立即退出
**解决方法**:
1. 检查端口占用
```bash
lsof -i :8080
```
2. 验证配置文件语法
```bash
tool validate-config
```

#### 问题3: 性能问题
**症状**: 响应缓慢，资源占用高
**解决方法**:
1. 调整配置参数
2. 启用缓存
3. 优化查询

### 日志查看
```bash
# 查看实时日志
tail -f /var/log/tool.log

# 搜索错误日志
grep -i error /var/log/tool.log

# 日志级别调整
tool --log-level debug
```
