> Elasticsearch是一个基于Lucene库的搜索引擎。它提供了一个分布式、支持多租户的全文搜索引擎，具有HTTP Web接口和无模式JSON文档

### 查询分词结果
#### 请求链接 /_analyze
#### 请求数据
```json
{
	"analyzer": "standard",
	"text": "我要去北京"
}
```

### 分页查询
```json
{
    "query": {
        "match_all": {}
    },
    "_source": [
        "id",
        "city",
        "loc"
    ],
    "from": 0,
    "size": 10
}
```
### 不进行分词搜索 term
```json
{
    "query": {
        "term": {
            "city": "NASHUA"
        }
    }
}
```
* 匹配多个词
```json
{
    "query": {
        "terms": {
            "city": ["NASHUA","NORTH WALTHAM"]
        }
    }
}
```
### 临近搜索词 match_phrase
```json
{
    "query": {
        "match_phrase": {
            "city": {
                "query": "NA UA",
                "slop": 1
            }
        }
    }
}
```
### 操作符 operator
or 分词检索 and 整个词搜索
```json
{
    "query": {
        "match": {
            "city": {
                "query": "NORTH WALTHAM",
                "operator": "and"
            }
        }
    }
}
```

### 根据id查询数据 ids
```json
{
    "query": {
        "ids": {
            "type": "_doc",
            "values": [
                "WlAx1IkB6VC3py0bNIPT",
                "W1Ax1IkB6VC3py0bNIPT"
            ]
        }
    }
}
```
