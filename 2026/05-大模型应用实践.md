## API调用指南

### OpenAI API
```python
from openai import OpenAI
import os

client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

response = client.chat.completions.create(
    model="gpt-4",
    messages=[
        {"role": "system", "content": "你是一个专业的技术顾问"},
        {"role": "user", "content": "请解释什么是RAG技术"}
    ],
    temperature=0.7,
    max_tokens=1000
)

print(response.choices[0].message.content)
```

### Anthropic Claude API
```python
import anthropic

client = anthropic.Anthropic(api_key=os.getenv("ANTHROPIC_API_KEY"))

message = client.messages.create(
    model="claude-3-opus-20240229",
    max_tokens=1000,
    messages=[
        {"role": "user", "content": "请解释什么是RAG技术"}
    ]
)

print(message.content[0].text)
```

### 流式输出
```python
from openai import OpenAI

client = OpenAI()

for chunk in client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "写一个Python快速排序"}],
    stream=True
):
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="")
```

## 本地部署

### Ollama部署

## 应用架构

### RAG架构
```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   用户输入   │────>│  向量数据库  │────>│  大模型API  │
└─────────────┘     └─────────────┘     └─────────────┘
                          │                   │
                          v                   v
                   ┌─────────────┐     ┌─────────────┐
                   │ 文档向量化  │     │  生成回答   │
                   └─────────────┘     └─────────────┘
```

### Agent架构
```python
class Agent:
    def __init__(self, llm, tools):
        self.llm = llm
        self.tools = tools
    
    def run(self, task):
        # 1. 分析任务
        plan = self.llm.plan(task)
        
        # 2. 执行工具
        for step in plan:
            result = self.execute_tool(step)
            
        # 3. 生成最终回答
        return self.llm.generate(task, result)
```

### 多模型协作
```python
class ModelOrchestrator:
    def __init__(self):
        self.gpt4 = OpenAIClient("gpt-4")
        self.claude = AnthropicClient("claude-3")
        self.llama = OllamaClient("llama2")
    
    def route(self, task):
        # 根据任务类型选择模型
        if task.complexity == "high":
            return self.gpt4.execute(task)
        elif task.type == "coding":
            return self.llama.execute(task)
        else:
            return self.claude.execute(task)
```

## 微调技术

### LoRA微调
```python
from peft import LoraConfig, get_peft_model
from transformers import AutoModelForCausalLM

model = AutoModelForCausalLM.from_pretrained("base_model")

lora_config = LoraConfig(
    r=8,
    lora_alpha=16,
    target_modules=["q_proj", "v_proj"],
    lora_dropout=0.05
)

model = get_peft_model(model, lora_config)
model.train()
```

## 最佳实践

### 1. 成本控制
- 使用合适的模型（不必总是用GPT-4）
- 设置max_tokens限制
- 实现缓存机制

### 2. 性能优化
- 异步API调用
- 并发请求处理
- 结果缓存

### 3. 安全建议
- 不在客户端暴露API密钥
- 实现请求验证
- 过滤敏感信息

### 4. 可靠性
- 实现重试机制
- 添加超时控制
- 监控API使用量
