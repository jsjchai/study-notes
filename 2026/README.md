## 笔记规范

### 文件命名
- 使用中文或英文描述性名称
- 格式：`序号-主题-描述.md`
- 避免特殊字符和空格

### 内容结构
1. **标题** - 清晰描述主题
2. **概述** - 简要介绍内容
3. **目录** - 使用Markdown TOC
4. **正文** - 分章节详细说明
5. **总结** - 关键要点回顾
6. **参考资料** - 相关链接和资源

### 标签系统
使用以下标签分类笔记：
- `#tutorial` - 教程类
- `#project` - 项目实践
- `#theory` - 理论知识
- `#practice` - 练习代码
- `#reference` - 参考资料
