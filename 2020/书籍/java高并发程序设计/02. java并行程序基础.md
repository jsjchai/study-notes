## 进程
* 系统进行资源分配和调度的基本单位，是操作系统结构的基础
* 线程就是轻量级进程，是程序执行的最小单位
## 线程状态
* NEW
* RUNNABLE
* BLOCKED
* WAITING
* TIMED_WAITING
* TERMINATED
## 线程的基本操作
1. 新建线程
2. 终止线程
  * stop()方法过于暴力，强行终止执行一半的线程，可能导致数据不一致
  * Thread.stop()结束线程时，会直接终止线程，并立即释放这个线程所持有的锁，这些锁恰好是用来维持对象一致性的
3. 线程中断
  * Thread.interrupt()
  * 目标线程中断，设置中断标志位
4. 等待与通知
  * wait() 当前线程就会在这个对象上等待
  * notify() 
5. 等待线程结束（join）和谦让（yield）
